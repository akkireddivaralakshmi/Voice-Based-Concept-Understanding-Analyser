# Voice-Based-Concept-Understanding-Analyser

An AI-powered educational assessment platform that evaluates conceptual understanding and speech fluency using speech-to-text transcription, semantic similarity analysis, audio processing, and intelligent scoring mechanisms.

## Live Demo

https://voice-based-concept-understanding-analyser.streamlit.app/

## Features

- Speech-to-Text Transcription using OpenAI Whisper
- Semantic Similarity Analysis using Sentence Transformers
- Audio Feature Extraction (Pause Ratio and RMS Energy)
- Filler Word Detection and Fluency Analysis
- Intelligent Understanding Scoring Engine
- Downloadable PDF Evaluation Reports
- Audio Waveform Visualization
- Streamlit Cloud Deployment
- Responsive User Interface
- Real-Time Concept Understanding Assessment

## Technologies Used

- Python
- Streamlit
- OpenAI Whisper
- Sentence Transformers
- Scikit-Learn
- Librosa
- NumPy
- Matplotlib
- FPDF2

## Workflow

1. Upload an audio explanation (.wav or .mp3)
2. Convert speech into text using Whisper
3. Compare the explanation with the reference concept
4. Extract audio features and detect filler words
5. Calculate the understanding score
6. Generate evaluation metrics
7. Generate and download the PDF report
